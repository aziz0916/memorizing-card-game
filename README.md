# 翻牌遊戲
此專案提供玩家進行翻牌遊戲，一次只能翻開兩張牌，若兩張牌點數一樣，則為配對成功

## Features - 專案功能
1. 一副撲克牌共有 52 張牌，分別為黑桃、紅心、方塊、梅花 4 種花色，每種花色有 13 張牌，分別為 1∼13 點
2. 遊戲開始時，牌桌上覆蓋 52 張牌，玩家需要翻開牌面才能得知點數
3. 玩家一次只能翻開兩張牌，若兩張牌點數一樣，則為配對成功，配對成功的牌可以保留翻開的狀態
4. 若配對失敗，則卡片會重新覆蓋，玩家要趁機記住卡片的號碼，尋求下一次配對成功的機會
5. 每成功配對一次可以得到 10 分，累積 260 分即達成任務
